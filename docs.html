Quick Start Guide

Step 1: Install the SDK

pip install gpuaas

Step 2: Get Your API Key

Navigate to your dashboard and generate an API key from Settings.

ðŸ’¡ Tip: Store your API key securely and never commit it to version control.

Step 3: Launch Your First Instance

from gpuaas import Client

client = Client(api_key="your-api-key")

# Launch a GPU instance
instance = client.instances.create(
    gpu_type="A100",
    image="pytorch:latest",
    region="us-east-1"
)

print(f"Instance created: {instance.id}")

GPU Instances

GPU instances are virtual machines with NVIDIA GPUs for AI workloads. Each instance includes CPU, RAM, storage, and network resources.

Available GPU Types

  • H200 SXM: 141GB HBM3e - For large-scale training
  • H100 SXM: 80GB HBM3 - LLM training
  • A100 SXM: 80GB HBM2e - Training & inference
  • L40S: 48GB GDDR6 - Inference
